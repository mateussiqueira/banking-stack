# Post 4: RFCs — Technical Depth Beyond Code

## Hook
Código sem RFCs é como navio sem mapa. Criei 3 RFCs técnicas para o Banking Challenges.

## Body

O Banking Challenges não é só código. São 3 RFCs que demonstram pensamento técnico profundo:

**1. Credit on Pix**
Como adicionar uma camada de crédito sobre o Pix?
- Sistema de avaliação de crédito em tempo real
- Parcelamento com cálculo de juros
- Integração com SPI para processamento
- Schema completo com Mermaid ERD

**2. Data Lake para Fintech**
Como ingerir dados de múltiplas fontes para análise?
- Arquitetura Medallion (Bronze/Silver/Gold)
- MinIO como storage S3-compatible
- Query layer com Presto/Trino
- Dashboards com Metabase

**3. Financial Monitoring**
Como monitorar transações em tempo real?
- Rule engine para detecção de anomalias
- Stream processing com Kafka/Redis
- Alerting via email, Slack, webhook
- Dashboards operacionais

Cada RFC inclui:
- Contexto do problema
- Arquitetura proposta
- Schema de dados (Mermaid)
- API design com exemplos
- Trade-offs e alternativas
- Questões em aberto

Isso demonstra capacidade de planejamento técnico, não só implementação.
