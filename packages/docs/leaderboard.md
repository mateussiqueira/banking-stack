Ranking dos alunos mais dedicados do Banking Challenges. Dados baseados no seu progresso local.

---

## Como funciona

Cada acao gera XP:

| Acao | XP |
|---|---|
| Completar um desafio | 100 XP |
| Acertar questao de quiz | 10 XP |
| Salvar um bookmark com nota | 5 XP |
| Bonus de streak (a cada 3 dias) | 25 XP * dias |

Seu nivel e calculado como `sqrt(XP / 50) + 1`.
