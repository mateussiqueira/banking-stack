**Formato:** Presencial (SP/RJ) + transmissão ao vivo
**Duração:** 2 horas
**Frequência:** Mensal
**Público:** Alunos, ex-alunos, mentores, empresas parceiras

---

## Formato do Evento (2h)

```
18:30 - 19:00  Credenciamento e network informal
19:00 - 19:10  Abertura e boas-vindas
19:10 - 19:40  Palestra técnica principal (30 min)
19:40 - 19:50  Perguntas e respostas
19:50 - 20:00  Pausa / networking
20:00 - 20:30  Painel de discussão (3-4 painelistas)
20:30 - 20:45  Lightning talks (3 talks de 5 min)
20:45 - 21:00  Encerramento e networking final
```

## Diretrizes para Palestrantes

### Antes do Evento

- [ ] Enviar slides com 14 dias de antecedência
- [ ] Confirmar necessidades técnicas (projetor, internet, microfone)
- [ ] Enviar bio e foto para divulgação
- [ ] Revisar tempo (máximo 30 min apresentação + 10 min Q&A)

### Durante a Apresentação

| Regra | Detalhe |
|-------|---------|
| Tempo | Máximo 30 minutos |
| Slides | Máximo 20 slides |
| Código | Demonstrações ao vivo aceitas, mas ter backup |
| Q&A | Reservar 10 minutos no final |
| linguagem | Técnica, mas acessível para iniciantes |

### Conteúdo Recomendado

**Temas que performam bem:**
- "Como resolvemos X problema em produção"
- "Migração de stack: lições aprendidas"
- "Arquitetura de sistema real (case study)"
- "Comparação técnica: Go vs Rust vs [X]"

**Temas a evitar:**
- Pura venda de produto/serviço
- Introdução genérica semCases práticos
- Slides apenas com texto (sem diagramas/código)

## Checklist de Implementação

### Pré-Evento (2 semanas antes)
- [ ] Confirmar palestrante e tema
- [ ] Reservar espaço (capacidade mínima: 80 pessoas)
- [ ] Criar página de inscrição
- [ ] Divulgar nas redes sociais
- [ ] Enviar convites para empresas parceiras
- [ ] Preparar material de credenciamento

### Durante o Evento
- [ ] Chegar 1h antes para montagem
- [ ] Testar projetor, microfone e internet
- [ ] Preparar kit do participante (crachá, QR code)
- [ ] Gravar transmissão ao vivo
- [ ] Coletar feedback (formulário rápido)

### Pós-Eventos (até 72h)
- [ ] Enviar emails de follow-up
- [ ] Publicar gravação no YouTube
- [ ] Compartilhar fotos nas redes
- [ ] Analisar métricas de participação
- [ ] Agendar próximo meetup
- [ ] Agradecer patrocinadores
