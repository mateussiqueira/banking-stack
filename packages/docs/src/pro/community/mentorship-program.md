# Programa de Mentoria - Banking Stack Pro

## Visão Geral

Programa de mentoria individual para acelerar carreira em engenharia de software, com foco em banking, fintechs e sistemas de alta performance.

| Item | Detalhes |
|------|----------|
| Duração | 3 meses |
| Encontros | 2x por mês (1h cada) |
| Total | 6 encontros |
| Formato | Videochamada + acompanhamento assíncrono |
| Vagas | 10 mentores por turma |

---

## Requisitos para Participar

### Do Mentor

- Experiência mínima de 3 anos em engenharia
- Trabalhou em projetos de banking/fintech ou sistemas distribuídos
- Disponibilidade de 2h por mês (encontros + preparação)
- Compromisso com 3 meses completos

### Do Aluno (Mentee)

- Completar Módulo 1 do Banking Stack Pro
- Ter projeto pessoal em andamento ou iniciado
- Disponibilidade para encontros quinzenais
- Compromisso de 3 meses

---

## Processo de Seleção

### 1. Inscrição (Semana 1)
- Formulário com:
  - Perfil do GitHub
  - Objetivos de carreira
  - Área de interesse (backend, infra, segurança, etc.)
  - Disponibilidade de horário

### 2. Matching (Semana 2)
- Sistema de compatibilidade por:
  - Interesses técnicos
  - Disponibilidade de horário
  - Nível de experiência
- Notificação por email

### 3. Kickoff (Semana 3)
- Reunião inicial com mentor e mentee
- Definição de objetivos para os 3 meses
- Combinar horários dos encontros

---

## Estrutura dos Encontros

### Formato Padrão (1h)

| Bloco | Duração | Conteúdo |
|-------|---------|----------|
| Check-in | 5 min | Como vai, novidades |
| Progresso | 15 min | Revisão de ações anteriores |
| Foco | 30 min | Tema principal do encontro |
| Próximos passos | 5 min | Ações para próxima sessão |
| Wrap-up | 5 min | Feedback e encerramento |

### Agenda Sugerida por Mês

#### Mês 1: Fundamentos e Carreira
- **Encontro 1**: Apresentação, objetivos, roadmap de carreira
- **Encontro 2**: Review do projeto pessoal, dicas de GitHub

#### Mês 2: Projetos e Networking
- **Encontro 3**: Arquitetura e design do projeto
- **Encontro 4**: Networking, como se destacar em entrevistas

#### Mês 3: Consolidação
- **Encontro 5**: Preparação para entrevistas técnicas
- **Encontro 6**: Review final, próximos passos, encerramento

---

## Temas de Mentoria

### Carreira
- Roadmap de carreira em engenharia
- Como se preparar para entrevistas
- Negociação salarial
- Transição de carreira

### Projetos Pessoais
- Escolha de stack tecnológica
- Arquitetura e design patterns
- Deploy e DevOps básico
- Como documentar projetos

### Networking
- Como construir presença online
- Contribuições open source
- Participação em comunidades
- Conferências e eventos

### Habilidades Técnicas
- Code review e boas práticas
- Testes e qualidade de código
- Performance e otimização
- Segurança em aplicações financeiras

---

## Acompanhamento

### Para o Mentee

**Antes de cada encontro:**
- Preencher formulário de progresso
- Enviar dúvidas e temas para discutir
- Atualizar repositório do projeto

**Depois de cada encontro:**
- Resumo das ações combinadas
- Feedback sobre a sessão
- Update no formulário de progresso

### Para o Mentor

**Antes de cada encontro:**
- Revisar formulário do mentee
- Preparar material de apoio
- Identificar oportunidades

**Depois de cada encontro:**
- Enviar resumo por email
- Anotar pontos para próxima sessão
- Atualizar dashboard de mentoria
