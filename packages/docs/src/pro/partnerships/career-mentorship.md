# Programa de Mentoria de Carreira — Banking Stack Pro

**Duração:** 3 meses
**Formato:** 1:1 semanal + workshops em grupo
**Público-alvo:** Alunos Premium que concluíram pelo menos 1 módulo

---

## Estrutura do Programa

### Mês 1: Fundação e Diagnóstico

| Semana | Atividade | Entregável |
|--------|-----------|------------|
| 1 | Sessão de kickoff — definição de objetivos | Plano de carreira personalizado |
| 2 | Análise de perfil técnico e gaps | Documento de diagnóstico |
| 3 | Revisão de currículo e LinkedIn | Currículo otimizado |
| 4 | Estratégia de busca ativa | Lista de 20 empresas-alvo |

### Mês 2: Desenvolvimento e Prática

| Semana | Atividade | Entregável |
|--------|-----------|------------|
| 5 | Preparação para entrevistas técnicas | Repo de exercícios |
| 6 | Simulação de entrevista (whiteboard) | Feedback estruturado |
| 7 | Construção de portfolio técnico | 3 projetos GitHub destacados |
| 8 | Networking estratégico | 10 conexões qualificadas |

### Mês 3: Execução e Fechamento

| Semana | Atividade | Entregável |
|--------|-----------|------------|
| 9 | Processos seletivos ativos | Pipeline de candidaturas |
| 10 | Negociação salarial e propostas | Checklist de negociação |
| 11 | Últimas revisões e ajustes | Currículo final |
| 12 | Sessão de encerramento | Plano de desenvolvimento contínuo |

## Sessões 1:1 Semanais

### Formato Padrão (60 min)

```
[0-5 min]  Check-in — como foi a semana?
[5-20 min] Revisão de ações da semana anterior
[20-40 min] Tópico principal da semana
[40-55 min] Próximos passos e compromissos
[55-60 min] Feedback e wrap-up
```

### Horários Disponíveis
- Segunda a Sexta: 08:00-12:00 / 14:00-18:00
- Sábado: 09:00-13:00
- Formato: Videochamada (Google Meet / Zoom)

## Revisão de Currículo

### Checklist de Revisão

- [ ] **Resumo profissional** (3-4 linhas focadas em fintech)
- [ ] **Skills técnicas** com níveis de proficiência
- [ ] **Projetos relevantes** (links GitHub com READMEs)
- [ ] **Experiência** com métricas (latência, throughput, disponibilidade)
- [ ] **Certificações** Banking Stack Pro destacadas
- [ ] **Formato** ATS-friendly (PDF, sem colunas complexas)

### Template de Resumo

```
Engenheiro de Software com [N] anos de experiência em [STACK].
Especializado em [DOMÍNIO] com foco em [TECNOLOGIAS].
[CONQUISTA MENSURÁVEL]. Banking Stack Pro Certified.
```

### Onde Destacar
- LinkedIn headline: "Go/Rust Engineer | Fintech | Banking Stack Pro Certified"
- LinkedIn about: Versão expandida do resumo
- GitHub: Pinned repos com projetos Banking Stack

## Preparação para Entrevistas

### Estrutura da Entrevista Técnica (Padrão Fintech)

```
1. Introdução (5 min)
2. Perguntas comportamentais (10 min)
3. Desafio técnico ao vivo (30 min)
4. Perguntas do candidato (10 min)
5. Próximos passos (5 min)
```

### Banco de Perguntas por Categoria

#### Comportamentais
- "Conte sobre um bug crítico que você resolveu em produção"
- "Como você lida com prazos apertados em sistemas financeiros?"
- "Descreva uma situação onde você precisou decidir entre qualidade e velocidade"

#### Técnicas (Go/Rust)
- "Como você implementaria idempotência em uma API de pagamentos?"
- "Explique o modelo de memória do Rust e por que é importante para sistemas financeiros"
- "Como resolveria um problema de race condition em um motor de transações?"

#### Sistemas
- "Projete um sistema de pagamentos que processe 10k transações/segundo"
- "Como você garantiria exactly-once processing em um pipeline Kafka?"
- "Arquitetura de alta disponibilidade para um gateway de pagamentos"

### Simulação de Entrevista

Cada aluno terá:
- 1 sessão de prática com mentor (formato real)
- Gravação para auto-análise
- Feedback escrito em 24h
