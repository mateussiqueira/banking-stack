## Programa de Colocação Profissional

### Fase 1: Pré-qualificação (Semana 1-2)
1. Aluno completa certificação Banking Stack Pro
2. Portfolio técnico revisado (projetos GitHub)
3. Entrevista técnica simulada com mentores

### Fase 2: Matching (Semana 3)
1. Currículo publicado no portal de vagas parceiro
2. Matching com vagas disponíveis nas empresas parceiras
3. Introdução via email para hiring managers

### Fase 3: Processo Seletivo (Semana 4-6)
1. Entrevista técnica simplificada (1 etapa ao invés de 3-4)
2. Código review do projeto final do Banking Stack
3. Fit cultural com time

### Fase 4: Onboarding (Semana 7-8)
1. Mentoria de integração com engenheiro sênior
2. Acompanhamento nos primeiros 90 dias
3. Feedback loop com Banking Stack Pro
