# ADR-005: Use Next.js for Dashboard

## Status

Accepted

## Context

Need modern web dashboard with SSR/SSG capabilities for the banking platform. Requirements include fast page loads, SEO-friendly content, and a rich interactive experience for users.

## Decision

Use Next.js 14 with App Router.

## Consequences

- Good developer experience with App Router
- Server-side rendering and static site generation
- SEO-friendly with automatic optimization
- Large React ecosystem for UI components
- Good performance with automatic code splitting
