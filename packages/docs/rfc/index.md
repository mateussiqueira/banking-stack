## Purpose

These RFCs document the thinking process behind major architectural decisions. Each includes:

- Problem statement and context
- Proposed solution with architecture diagrams
- Database schema (Mermaid ERD)
- API design with examples
- Trade-offs and alternatives
- Security considerations
- Open questions
